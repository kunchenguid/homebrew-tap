# Kun's Homebrew Tap

This repository is a general Homebrew tap for Kun Chen's apps and developer tools.

## Baby Menu

Install Baby Menu with:

```sh
brew install --cask kunchenguid/tap/baby-menu
```

Upgrade Baby Menu with:

```sh
brew update
brew upgrade --cask baby-menu
```

Baby Menu stores mutable app state in `~/.baby-menu` so generated extensions survive app upgrades.
